3. Backtesting

In this section, we will implement and backtest a Bollinger Bands Mean Reversion strategy for each stock.

(1) Generate signals and positions that follows the Bollinger Bands Mean Reversion Strategy.

In[49]:
def Bollinger_strategy(dfi, window=20):
    sma = dfi.rolling(window).mean()
    upper = sma + 2 * dfi.rolling(window).std()
    lower = sma - 2 * dfi.rolling(window).std()

    # Generate Entry/Exit buy/sell signals
    position = 0
    positions = []
    for i in range(len(dfi)):
        if dfi.iloc[i] < lower.iloc[i] and position == 0:
            position = 1  # Enter long
        elif dfi.iloc[i] > upper.iloc[i] and position == 0:
            position = -1  # Enter short
        elif position == 1 and dfi.iloc[i] > sma.iloc[i]:
            position = 0  # Exit long
        elif position == -1 and dfi.iloc[i] < sma.iloc[i]:
            position = 0  # Exit short
        positions.append(position)
    return pd.Series(positions, index=dfi.index)

(2) Calculate the simple daily returns. And then, apply the Bollinger strategy that we have defined above to calculate the strategy daily returns.

In[50]:
# Apply the Bollinger Bands strategy to each ticker
strategy_returns = {}
buyhold_returns = {}

for ticker in df.columns:
    prices = df[ticker]
    positions = Bollinger_strategy(prices)
    # Calculate simple daily returns and strategy returns
    daily_returns = prices.pct_change().fillna(0)
    # Shift positions in order to avoid lookahead bias
    strat_returns = daily_returns * positions.shift().fillna(0)
    
    # Store the results in dictionaries
    strategy_returns[ticker] = strat_returns
    buyhold_returns[ticker] = daily_returns

5. Portfolio Optimisation

(1) Implement simple portfolio optimisation by maximising the Sharpe Ratio.

In[56]:
# First we assume a risk-free rate of 4.41% (0.0441) and 252 trading days in a year
RFR = 0.0441
TRADING_DAYS = 252

def negative_sharpe(weights):
    # Calculate annualised portfolio return
    p_returns = np.dot(buyhold_returns_df.mean(), weights) * TRADING_DAYS
    # Calculate annualised portfolio volatility
    cov = buyhold_returns_df.cov() * TRADING_DAYS
    p_volatility = np.sqrt(np.dot(weights, np.dot(cov, weights)))
    sharpe_ratio = (p_returns - RFR) / p_volatility
    return -sharpe_ratio  # Return the negative Sharpe ratio for minimization

# Constraints
cons = {"type": "eq", "fun": lambda x: sum(x) - 1}

# Bounds
bounds = tuple((0, 1) for x in range(10))

# Weights
init_weights = np.ones(10) / 10

# Optimisation
optimal = minimize(negative_sharpe, init_weights, bounds=bounds, constraints=cons)
sharpe_weights = optimal.x

print(f"Optimised Sharpe Ratio: {optimal.fun:.4f}")

print("Optimised Portfolio Weights:")
for ticker, weight in zip(df.columns, sharpe_weights):
    print(f"{ticker}: {weight:.4f}")

(2) Implement an AI-generated portfolio optimisation

According to the results AI generated, I chose Minimum Variance Portfolio

Objective: Minimize portfolio variance.

Metric: Portfolio variance (risk).

Use case: Risk-averse investors.

Constraints: Sum of weights = 1, weights 0 (long-only).

The Minimum Variance Portfolio strategy aims to allocate capital in a way that minimises the portfolios overall risk, as measured by variance. The core principle is to utilise the diversification effect to combine each asset into a more stable investment portfolio than a single asset. This method is especially applicable to risk-averse investors since it minimises the volatility. It does not rely on return prediction and only focuses on minimizing risks. Therefore, it has strong robustness and wide application value.

def portfolio_variance(weights):
    # Calculate annualized covariance matrix
    cov = buyhold_returns_df.cov() * TRADING_DAYS
    # Calculate portfolio variance (scalar)
    variance = np.dot(weights, np.dot(cov, weights))
    return variance

6. Reporting

Based on the backtest and portfolio optimisation results conducted for 20202024, I recommend adopting the Sharpe Ratio Optimised Portfolio as the primary allocation strategy. This approach significantly outperforms both the minimum variance and equal-weight portfolios in terms of cumulative returns. It can be seen from the summary table that the optimal weights identified by the Sharpe ratio optimisation method are highly concentrated in NVDA (50.95%), TSLA (25.22%), SMCI (19.64%), and IAG (4.19%). All the rest of the stocks weigh zero, indicating that they have lower risk-adjusted returns over the backtest period. Although the minimum variance and equal-weight portfolios are more diversified, their cumulative returns are much lower than the Sharpe-optimised portfolio. This outcome is primarily due to their portfolio construction method. For instance, the equal weight portfolio assigns a fixed 10% to each stock, including many underperformers. Therefore, these stocks dilute the impact of top-performing assets. The minimum variance portfolio focuses solely on reducing overall portfolio volatility. This leads to higher weights for low-risk stocks while ignoring their returns. In a word, my recommended investment portfolio is NVDA (50.95%), TSLA (25.22%), SMCI (19.64%), and IAG (4.19%).
